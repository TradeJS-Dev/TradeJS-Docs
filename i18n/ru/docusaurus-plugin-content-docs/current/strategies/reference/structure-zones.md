---
title: 'StructureZones'
---

`StructureZones` - встроенная TypeScript-стратегия из `@tradejs/strategies`.

Она исследует market-structure zones и имеет strategy-specific AI adapter.

Используйте для research:

- support/resistance structure;
- zone-based entries;
- chart figure inspection;
- AI-gated runtime experiments.

Держите zone detection causal и проверяйте chart artifacts перед выводами по метрикам.
