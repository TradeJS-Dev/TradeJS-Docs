---
title: 'ReverseTrendLine'
---

`ReverseTrendLine` - встроенная TypeScript-стратегия из `@tradejs/strategies`.

Она относится к trendline workflow, но ориентирована на reverse/rejection behavior, а не основной breakout path.

Используйте для research:

- trendline rejection setups;
- support/resistance geometry;
- chart figures inspection;
- сравнение с `TrendLine`.

Проверяйте поведение через backtests и figures. Геометрия trendline - input для анализа, а не гарантия результата.
