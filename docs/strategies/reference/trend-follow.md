`TrendFollow` is a built-in TypeScript strategy from `@tradejs/strategies`.

It researches continuation-style trend setups with strategy-specific deterministic AI guardrails.

## Use It For Researching

- trend continuation
- regime filters
- stop-distance filters
- derivatives flush context

## Notes

Recent source commits tightened TrendFollow gates around beta, stop distance, market regime, and derivatives context. Treat those as implementation details to validate, not as universal defaults.
