title: 'StructureZones'

`StructureZones` is a built-in TypeScript strategy from `@tradejs/strategies`.

It researches market-structure zones and exposes a strategy-specific AI adapter.

## Use It For Researching

- support/resistance structure
- zone-based entries
- chart figure inspection
- AI-gated runtime experiments

## Notes

Keep zone detection causal and inspect chart artifacts before trusting metrics.
